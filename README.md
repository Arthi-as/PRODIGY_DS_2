# Project Documentation

This project contains task-related code and notebooks for data analysis and processing.

## Files

- `prodigy_task2.ipynb` - Main Jupyter notebook for task 2

## Setup

To get started with this project, follow the steps below:

1. Clone or navigate to the project directory
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter notebook to view and run the analysis

See `requirements.txt` for a list of Python packages needed to run this project.
